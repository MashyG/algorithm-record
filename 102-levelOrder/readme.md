## 题目

[二叉树的层序遍历](https://leetcode.cn/problems/binary-tree-level-order-traversal/)

> 描述：给你二叉树的根节点 root ，返回其节点值的 层序遍历 。 （即逐层地，从左到右访问所有节点）。

- - -

### 示例1

![示例1](./assets/tree1.jpg)

```
输入：root = [3,9,20,null,null,15,7]
输出：[[3],[9,20],[15,7]]
```


### 示例2

```
输入：root = [1]
输出：[[1]]
```

### 示例3

```
输入：root = []
输出：[]
```
